This isn't another todo-app or weather app. It's a **production-grade ML system** that:

- **Detects market manipulation** in real-time
- **Uses Isolation Forest algorithm** (same as scikit-learn)
- **Streams live data** from Binance exchange
- **Generates smart alerts** with severity scoring
- **Zero dependencies** - pure vanilla JavaScript
